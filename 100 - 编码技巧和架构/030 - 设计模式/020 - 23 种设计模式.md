## 建造者模式

和工厂不同的是，工厂模式入参通常需要一个 code 或一个类型就能返回一个对象，对象的具体类是什么根据入参而定

![[../../020 - 附件文件夹/Pasted image 20230422183006.png]]

但建造者模式是用来构建复杂对象，举个买电脑的例子

工厂模式就像是，我告诉工厂我要联想的电脑还是华硕的电脑，工厂就会给我一个对应品牌和型号的电脑

建造者模式就像是，我告诉建造者我的电脑要三星 32G 内存条，13 代 i7 的CPU，长江存储的 1T 固态，建造者根据生产电脑的模板流程但每一步的选材都是有客户端自定义的

所以我对建造者模式的理解是：建造者 = 模板模式 + 有更多入参的工厂模式，并用于构建复杂对象
